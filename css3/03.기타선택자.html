1. 속성 선택자

선택하고자 하는 요소 내에 작성되어있는 속성을 이용하여 선택하는 방법
선택자 뒤에 []를 이용하여 속성과 속성값을 제시하면서 선택 

    선택자[속성=속성값] {스타일속성 : 값;}    => 속성 = 속성값일 때
    선택자[속성~=속성값] {스타일속성 : 값;}   => 속성값이 단어형태로 포함되어 있을 때         | ' ' 공백으로 구분
    선택자[속성|=속성값] {스타일속성 : 값;}   => 속성값이 특정 값을 단어로 포함하고 있을 때   | '-' 대시로 구분
    선택자[속성^=속성값] {스타일속성 : 값;}   => 속성값으로 시작하는 태그 선택
    선택자[속성$=속성값] {스타일속성 : 값;}   => 속성값으로 끝나는 태그 선택
    선택자[속성*=속성값] {스타일속성 : 값;}   => 속성값이 포함되어있는 모든 태그

2. 자손선택자 후손선택자

요소들이 중첩되어 작성 가능 자손 : 바로 하위인 요소들 후손 : 하위 요소들 전부 

    * 자손 선택자 : >
      a > b {
        스타일속성 : 값;
      }
    
    * 후손 선택자 : ' '(공백)
      a b {
        스타일속성 : 값;
      }

3. 동위(같은 레벨) 선택자

동위관계(같은 레벨)에 위치한 특정 요소를 선택할 때 

    * a요소 다음에 있는 b요소 "하나만" 선택 시
      a + b {
        스타일속성 : 값;
      }
    
    * a요소 다음에 있는 b요소 "전부" 선택 시
      a ~ b {
        스타일속성 : 값;
      }

4. 반응 선택자

사용자에 움직임에 따라 선택되는 선택자 

    * 해당 요소를 클릭할 경우 스타일 부여
      선택자:active {
        스타일속성 : 값;
      }
    
    * 해당 요소에 마우스 오버 시
      선택자:hover {
        스타일속성 : 값;
      }

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자 

    1) 체크된 상태의 요소 내에
      선택자:checked {
        스타일속성 : 값;
      }
  

    2) 포커스된 상태의 input 요소 선택
      선택자:focus {
        스타일속성 : 값;
      }

    3) 활성화(enable) 되어있는 요소들 선택
      선택자:enabled {
        스타일속성 : 값;
      }
    4) 비활성화(disable) 되어있는 요소들 선택
      선택자:disabled {
        스타일속성 : 값;
      }

font-size : 글꼴 크기를 지정할 때 사용

    선택자 {
      font-size : (px|em|%);
    }
    - px : 픽셀단위 고정크기
    - em : 부모 요소 크기의 배수
    - rem : HTML 태그의 기본 글꼴 크기의 배수
    - % : 상위 요소 폰트크기의 비율로 계산

    - vw : 현재 보이는 창의 크기에 따라 값이 변한다(창이 1000px 이면 1vw = 10px)
    - vh : vw와 동일

    옵션들
      * 고정값
        xx-small : 10px
        x-small : 10px
        small : 13px
        medium : 16px
        large : 18px
        x-large : 24px
        xx-large : 32px

      * 상대값
        larger : 부모의 글자크기에 1/5를 더함( ~= 120%, 1.2em)
        smaller : 부모의 크기에 1/6을 뺌( ~= 87%, 0.87em)
  • 가변크기 px테스트
  • 가변크기 em테스트
  • 가변크기 rem테스트
  • 가변크기 %테스트

    }

  • 원래 굵기보다 더 가늘게

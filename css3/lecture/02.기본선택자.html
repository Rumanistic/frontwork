CSS 기본선택자

선택자란?

특정 html 요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택하여 원하는 스타일과 기능을 적용시킬 수 있음

1. 모든(전체) 선택자 : "*"

현재 이 문서의 모든 요소들을 선택하고자 할 때 사용 

    * {
      스타일속성 : 값;
      스타일속성 : 값;
      ...
    }

2. 태그 선택자 : "tagName"

현재 이 문서에 사용된 같은 태그명의 요소들을 선택하고자 할 때 사용 

    $tagName {
      스타일속성 : 값;
      스타일속성 : 값;
      ...
    }

3. 아이디 선택자 : "#idName"

현재 문서에서 특정한 엘리먼트를 선택해서 변경하고자 할 때,
id로 지정해서 선택 가능
단, 해당요소에 id속성을 이용하여 고유한 id값을 부여해야함 

    #idName {
      스타일속성 : 값;
      스타일속성 : 값;
      ...
    }

4. 클래스 선택자 : ".className"

현재 문서에 사용된 클래스명 중에서 원하는 태그의 클래스들을 선택하고자 할 때 사용 

    [대상이 되는 클래스가 속한 헤더] {
      스타일속성 : 값;
      스타일속성 : 값;
      ...
    }
    
    .className{
      스타일속성 : 값;
    }

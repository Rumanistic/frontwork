Content (innerHTML, innerText) 관련 메소드

* html('[문구]') 메서드 : innerHTML 속성과 관련된 메서드

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메서드
  > $('선택자').html()
    html() : 선택된 요소의 content영역 값을 반환해줌(tag 포함)
    html('문구') : 선택된 요소의 content영역 값을 문구로 대체함

* text('[문구]') 메서드 : innerText 속성과 관련된 메서드

선택된 요소의 content 영역의 값을 리턴 또는 변경해주는 메서드
  > $('선택자').text()
    text() : 선택된 요소의 content영역 값을 반환해줌(텍스트만)
    text('문구') : 선택된 요소의 content영역 값을 문구로 대체함

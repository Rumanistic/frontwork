추가적인 메서드

* each 메서드

배열의 모든 인덱스에 순차적으로 접근할 때 사용
객체가 가지고 있는 모든 속성에 순차적으로 접근하고자 할 때
for in문과 유사하게 수행되는 메서드 

    1) $.each([객체/배열], ([매개변수]) => {
        순차적으로 실행할 반복문
      })

    2) [객체/배열].each(() => {
        순차적으로 실행할 반복문
      })
    
    만약 객체를 제시했다면
        첫번째 매개변수에는 순차적으로 접근할 때 마다의 객체의 속성명(key)가 담김
        두번째 매개변수에는 해당 속성값(value)이 담김

    만약 배열을 제시했다면
        첫번째 매개변수에는 순차적으로 접근할 때 마다의 인덱스 번호가 담김
        두번째 매개변수에는 해당 인덱스의 값(value)이 담김

  1. [window.]alert("알림창에 출력할 문구");
  2. [window.]console.log("콘솔창에 출력할 문구");
  3. document.write("화면에 출력할 문구");
  4. 선택한요소.innerHTML|innerText="요소에 출력할 문구"

3) document.write

4) 선택한요소.innerHTML|innerText="요소에 출력할 문구"

변경하고자 할 때 innerHTML, innerText라는 속성에 접근해서 가능

2. 데이터 입력받는 구문(변수에 기록 가능)

1) 변수 = [window.]confirm("질문내용");
2) 변수 = [window.]prompt("질문내용");
3) 변수 = 선택한요소.속성(id,className,innerHTML,innerText, ...);
4) 변수 = 선택한input요소.value;

1) [window.]confirm();

confirm 호출 시 "질문내용"과 "확인/취소" 버튼이 존재하는 알림창 발생
확인 버튼 클릭 시 true, 취소 버튼 클릭 시 false 반환

2) [window.]prompt();

prompt 호출 시 "질문내용"과 입력할 수 있는 "텍스트 상자"와
"확인/취소" 버튼이 보여지는 알림창 발생
확인 버튼 클릭 시 텍스트 상자에 입력한 문자열 반환, 취소 버튼 클릭 시 null 반환

4) 선택한input요소.value;

이미지 관련 태그

* src 속성


* alt 속성

- 사진의 경로가 잘못되었거나, 이미지를 제대로 표현하지 못했을 때 대체 텍스트의 용도로 사용
- 또한 시각장애인들을 위한 스크린리더(화면낭독기) 에서 이미지를 읽어주는 설명 문구


* width와 height 속성

이미지의 가로, 세로 길이를 조절하는 속성
고정길이(px), 가변길이(%)으로 조정 => 기본값은 px

고정길이단위(px) === 기본값 : 화면 사이즈가 조정되어도 이미지의 크기는 변하지 않음


가변길이단위(%) : 화면 사이즈 혹은 부모요소 사이즈에 따라 이미지 크기 변동

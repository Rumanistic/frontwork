폼 관련 태그

form 태그 내에 submit 버튼 클릭 시 해당 form 안에 작성된
사용자가 입력한 정보를 서버로 전송하면서 요청하는 역할을 수행

form action="" method=""

* action 속성 : 해당 form에 사용자가 입력한 값들을
        전달하면서 요청할 서버의 경로
* method 속성 : 요청 전송 방식을 지정하는 속성 (get[default]|post)
       > get방식 : 요청 시 사용자가 입력한 값들이 url에 노출되는 방식
       > post방식 : 요청 시 사용자가 입력한 값들이 url에 노출되지 않는 방식


form 내의 submit 버튼 클릭 시 action 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨(사용자가 입력한 값들은 key=value 세트로 전달[json 형식])
search?id=kkk&pw=1234&age=24&...
